# FuryRoad RC Club Logo

Place your logo file here with one of these names:
- `logo.png` (recommended - PNG format)
- `logo.svg` (best quality - SVG format)
- `logo.jpg` (alternative - JPG format)

The logo will be automatically used throughout the application:
- Login page
- Sidebar header
- Dashboard
- Browser favicon

**Recommended specifications:**
- Size: 512x512 pixels or larger
- Format: PNG with transparency or SVG
- Background: Transparent (preferred) or white
- File size: Under 500KB for optimal performance
